Linux common command

  • ls 查看当先目录文件
  • mv
    • 重命名（rename）【file】or【directories】
    • 移动【file】to【directories】
  • cd 进入指定目录
  • touch create new file
  • mkdir create new directories
  • rm delete file
    • [-f] force
    • [-r] delete directories
